# EEG-Brain-Tumor-Detection-Hybrid-CNN-BiLSTM
A hybrid 1D-CNN–BiLSTM deep learning framework for non-invasive brain tumor detection using IoT-enabled EEG simulation. Achieves 72.5% accuracy, 0.8185 AUC on UCI Epileptic Seizure dataset (tumor vs. normal). Includes PCA/ICA preprocessing, 5‑fold CV, Grad‑CAM interpretability.
